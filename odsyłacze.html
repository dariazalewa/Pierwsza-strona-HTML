HTML i CSS - Odsyłacze (hiperłącza)


Rozdziały:
  • Powrót do strony głównej

    • Często na stronach internetowych mamy odsyłacze(hiperłącza) np. do innych stron, aplikacji, usług.Prowadzą one nas do innych stron lub dowolnych elementów bieżącej witryny.
    Definiowany odsyłacz ma postać :

    <a href="adres URL"> Tekst połączenia </a>

    Przykłady odsyłaczy:

    Do innych dokumentów<a href="plik.html"> Dokument </a>
    Do pliku z grafiką<a href="grafika.jpg"> Rysunek </a>
    Do innej strony internetowej<a href="https://www.facebook.com/"> Facebook </a>
    Do zasobu dostępnego przez FTP<a href="ftp://ftp.tpnet.pl"> TPNET </a>

    Przykład takiego hiperłącza:

    <a href="https://www.youtube.com/"> YouTube </a>

    Rezultat:

    YouTube

    Inny przykład, tym razem z rysunkiem:

    <a href="obraz.jpg"> Windows XP </a>

    Rezultat:

    Windows XP

    Tworzenie odsyłaczy do plików muzycznych:

    <a href="muzyka.mp3/"> Muzyka </a>

    <a href="film.avi/"> <img src="obraz.jpg" width:100px; height:100px; alt="wyświetlenie"> </a>

    <a href="https://www.youtube.com/watch?v=-lfHXKbsMLE/"> <img src="hyperlink.jpg" style="width:300px; height:200px;" alt="wyświetlenie"> </a>
