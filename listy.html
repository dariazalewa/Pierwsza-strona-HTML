HTML i CSS - Listy.


Rozdziały:
  • Powrót do strony głównej

    • Listy nieuporządkowane(wypunktowane) oznaczamy:

    <ul>
    :
    </ul>

    W znaczniku <ul> wpisujemy atrybut TYPE określający kształt wypunktowania, np.:

    <body>
    <ul TYPE="square">
    <li>pierwsza</li>
    </ul>

    Mamy 3 typy:

    Zamalowane kółko:

    • 1

    Niezamalowane kółko:

    • 2

    Kwadrat:

    • 3

    Listy uporządkowane(numerowane) oznaczamy:
    <ol>
    :
    </ol>

    Tutaj jednak atrybut TYPE określamy jako:

    TYPEOpis
    TYPE="1"Ponumerowane liczbami(domyślne).
    TYPE="A"Ponumerowane wielkimi literami.
    TYPE="a"Ponumerowane małymi literami.
    TYPE="I"Ponumerowane wielkimi cyframi rzymskimi.
    TYPE="i"Ponumerowane małymi cyframi rzymskimi.

    Jeśli chcemy rozpocząć wyliczanie od określonej wartości np. od 2 linii a nie od 1 to wpisujemu w znaczniku <ol> atrybut START="liczba":

    <ol START="2">
    <li>
    ...
    </li>
    </ol>

    Możemy też te dwie możliwości połączyć np. ponumerować małymi rzymskimi liczbami i zacząć wyliczać od 3:
    kod:

    <ol TYPE="i" START="3">
    <li>1 </li>
    <li>2 </li>
    <li>3 </li>
    </ol>

    rezultat:

    1. 1
    2. 2
    3. 3

    Ostatnią ważną rzeczą są listy definicji, które służą do tworzenia słowników, wyjaśnień i opisów różnych terminów.
    Kod składa się w następujący sposób:

    <DL>
    <DT>Język HTML </DT>
    <DD>...</DD>
    </DL>

    Znaczniki <DL>...</DL> zawierają część definiującą.
    Znaczniki <DT>...</DT> część definiowaną.
    A znaczniki <DD>...</DD> określają część definiowanej.
