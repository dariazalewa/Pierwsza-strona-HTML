Przekształcenia CSS(2D i 3D)

  • Główna

Przykład dla 2D:
Kod:
<div style="width: 100px; height: 50px; background-color: blueviolet; border: 1px solid black;"> Normalny div</div>
<div style="width: 100px; height: 50px; background-color: blueviolet; border: 1px solid black; transform: matrix(1, -0.3, 0, 1, 0, 0);">Zmieniony</div>
Rezultat:

Normalny div

Zmieniony

Dzięki właściwości transformacji CSS możesz użyć następujących metod transformacji 2D:

♦ translate() - przesuwa element zgodnie z parametrami podanymi dla osi X i Y, w przykładzie przesuwa element o 50 pikseli w prawo i 100 pikseli w dół od jego aktualnej pozycji,

Przykład: div {transform: translate(50px,100px);}

♦ rotate() - obraca element zgodnie z ruchem wskazówek zegara(wartości dodatnie) lub przeciwnie(wartości ujemne) zgodnie z określonym stopniem,

Przykład: div {transform: rotate(10deg);}
lub div {transform: rotate(-10deg);}

♦ scale() - zwiększa lub zmniejsza rozmiar elementu zgodnie z parametrami podanymi dla szerokości i wysokości, w przykładzie 4 to szerokość i 5 to wysokość,

Przykład: div {transform: scale(4, 5);}

♦ scaleX() - zwiększa lub zmniejsza szerokość elementu,

Przykład: div {transform: scaleX(6);}

♦ scaleY() - zwiększa lub zmniejsza wysokość elementu,

Przykład: div {transform: scaleY(9);}

♦ skew() - pochyla element wzdłuż osi X i Y o podane kąty, w przykładzie pochyla element o 40 stopni wzdłuż osi X i 30 stopni wzdłuż osi Y,

Przykład: div {transform: skew(40deg, 30deg);}

♦ skewX() - pochyla element wzdłuż osi X o podany kąt,

Przykład: div {transform: skewX(10deg);}

♦ skewY() -pochyla element wzdłuż osi Y o podany kąt,

Przykład: div {transform: skewY(30deg);}

♦ matrix() - łączy wszystkie metody transformacji 2D w jedną. Kod wygląda następująco: matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY()),

Przykład: div {transform: matrix(1, -0.3, 0, 1, 0, 0);}

Przekształcenia CSS umożliwiają przesuwanie, obracanie, skalowanie i pochylanie elementów.
