HTML i CSS - Arkusz stylów, czyli wprowadzenie do języka CSS


Rozdziały:
  • Powrót do strony głównej

    • Język CSS umożliwia tworzenie układu graficznego dokumentu HTML, np. wielkość czcionki, rodzaj czcionki, wysokość/szerokość obrazków, itp.
    Wszystkie polecenia dotyczącze formatowania powinny zostać umieszczone w oddzielnym pliku i powiązane z elementami kodu HTML(zewnętrzny arkusz stylów).
    Składnia kaskadowych arkuszy stylów:

    Selektor {
    właściwości:wartość;
    }

    Przykład:

    body {
    background color: blue;
    font-size: 12 px;
    }

    W dokumencie HTML istnieje kilka sposobów dołączenia stylów CSS:
    a) styl lokalny(w linii),
    b) wewnętrzny arkusz stylów,
    c) zewnętrzny arkusz stylów,

    a) W stylu lokalnym jest utworzony styl w jednej linii z elementem formatowania, np.
    < body style="background-color: red;">
    <p style="color: darkolivegreen;">
     Znacznik <span> służy do grupowania/wyróżniania kilku elementów, np. obrazów, słów, akapitów w celu nadania im określonego stylu.

    <span style="właściwości:wartość;">...</span>.

    Przykładowe użycie znacznika <span>:

    <h3> Mam obecnie <span style="color:brown;">praktyki</span> zawodowe.</h3>
