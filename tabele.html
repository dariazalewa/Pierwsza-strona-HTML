HTML i CSS - powtórzenie

Tabele

  • Główna

Kod, dzięki któremu powstanie nam tabela składa się w następujący sposób:

<table>
<tr> <th>...</th> <th>...</th></tr>
<tr> <td>...</td> <td>...</td></tr>
</table>

Znacznik <table>...<table> zaczyna tworzenie tabeli.
Znaczniki <th>...</th> są nagłówkami tabeli.
Znaczniki <tr>...</tr>to wiersze w tabeli.
Znaczniki <td>...</td>to zwykłe komórki w tabeli.

Przykład kodu:

<table>
<tr> <th>Nazwiko</th> <th>Imię</th></tr>
<tr> <td>Chmielewski</td> <td>Jan</td></tr>
</table>

Rezultat:

Nazwiko Imię
Chmielewski Jan

Aby poprawnie wyświetlało nam tabelki w części <style> w nagłówku <head> lub oddzielnym pliku CSS możemy wpisać np.:

table, td, th {
border: 1px solid rgb(165, 164, 164);
border-collapse: collapse;
}
th, td{
padding: 5px;
text-align: left;
}
