Edytory WYSIWYG

  • Główna

Edytory typu WYSIWYG (ang. What You See Is What You Get - otrzymujesz to, co widzisz) pozwalają uzyskać rezultat, że docelowa strona będzie wyglądać tak samo jak ta, którą tworzymy i edytujemy w jakimś edytorze.

Funkcje edytorów WYSIWYG

Edytory pozwalają nam na:

a) używanie różnych zdefiniowanych stylów dla tekstu - akapitu i nagłówków, które automatycznie formatują tekst zgodnie z formatowaniem określonym w projekcie graficznym strony - znaczenie ikon używanych w edytorze,

b) określanie wyrównanie tekstu - do lewej, do prawej, wyśrodkowany, wyjustowany,

c) zmienianie kolor tekstu,

d) wstawianie tabel, które mogą być przygotowane w edytorze lub importować dane z plików Excela (CSV),

e) wstawianie i kadrowanie ilustracji z Menadżera ilustracji,

f) wstawianie filmów lub plików audio z YouTube"a lub własnego serwera,

g) wstawianie kodu html oczywiście,

h) wstawianie znaków specjalny wybierając go z wielu dostępnych,

i) wstawianie łączników opcjonalnych, twardych spacji i linii podziału strony,

j) przywracanie wcześniejszych wersji dokumentów,

k) tworzenie szablonów i bibliotek,

l) weryfikację odsyłaczy,

m) wysyłanie strony lub witryny na serwer.

Dostępne edytory WYSIWYG

1.Adobe Dreamweaver - Dreamweaver to edytor WYSIWYG i edytor kodu, który obsługuje wszystko, co potrzeba do stworzenia zaawansowanych stron i aplikacji internetowych. Wspiera technologie CSS, JSP, XHTML, PHP, JavaScript, PHP5, ASP, ASP.NET, XML i AJAX. W jego skład wchodzą narzędzia do tworzenia stron internetowych, np. narzędzie do kontroli właściwości CSS czy do testowania stron w różnych przeglądarkach.

2.KompoZer - KompoZer to darmowy edytor WYSIWYG, dzięki któemu możemy tworzyć strony internetowe bez znajomości kodów HTML i CSS. Program umożliwia tworzenie strony zawierającej, np. tabele czy formularze. Posiada też wbudowany edytor kaskadowych arkuszy stylów CSS i klienta FTP.

3.Visual Web Developer 2017 Express Edition - jest to darmowe narzędzie przeznaczone do tworzenia aplikacji internetowych (ASP.NET) na platformie .NET Framework 3.5. Program ten ma edytor wizualny, rozbudowany edytor kodu, system inteligentnych podpowiedzi oraz debugger.

Dzięki temu tworzenie i edycja stron www jest wyjątkowo prosta i intuicyjna i nie wymaga znajomości kodu html.
