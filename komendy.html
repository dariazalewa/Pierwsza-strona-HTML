3. Element <canvas> służy do rysowania grafiki za pomocą JavaScript. Element <canvas> jest tylko pojemnikiem na grafikę, musimy użyć JavaScript, aby faktycznie grafikę narysować. Canvas ma kilka metod rysowania ścieżek, ramek, okręgów, tekstu i dodawania obrazów.
Przykład podstawego, pustego elementu canvas:
<canvas id="myCanvas" width="200" height="100" style="border:1px solid black;"></canvas>
Rezultat:

Teraz dodajemy JavaScript aby coś namalować:

a) Rysowanie linii:
<canvas id="myCanvas" width="200" height="100" style="border:1px solid white"></canvas>
<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.moveTo(0,0);
ctx.lineTo(200,100);
ctx.stroke();
</script>
Rezultat:

b) Rysowanie koła:
<canvas id="myCanvas" width="200" height="100" style="border:1px solid red"></canvas>
<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.beginPath();
ctx.arc(95,50,40,0,2*Math.PI);
ctx.stroke();
</script>
Rezultat:

c) Rysowanie tekstu:
<canvas id="myCanvas" width="300" height="80" style="border:1px solid gray"></canvas>
<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.font = "30px Arial";
ctx.fillText("Strona internetowa",10,50);
</script>
Rezultat:

d) Rysowanie obramowania tekstu:
<canvas id="myCanvas" width="300" height="80" style="border:1px solid gray"></canvas>
<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.font = "30px Arial";
ctx.strokeText("Strona internetowa",10,50);
</script>
Rezultat:

e) Rysuje gradient liniowy:
<canvas id="myCanvas" width="200" height="100" style="border:1px solid black"></canvas>
<script>
var c = document.getElementById("my");
var ctx = c.getContext("2d");
// Tworzymy gradient
var grd = ctx.createLinearGradient(0,0,200,0);
grd.addColorStop(0,"blue");
grd.addColorStop(1,"white");
// Wypełnij gradient
ctx.fillStyle = grd;
ctx.fillRect(10,10,150,80);
</script>
